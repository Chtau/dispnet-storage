# dispnet-storage

Storage library used in dispnet.

## Build instruction

### 1.) Install [Rust](https://www.rust-lang.org/tools/install)

### 2.) Build

```sh
cargo build
```

### .) Test

```sh
cargo test
```

### .) Benchmark

```sh
cargo bench
```
